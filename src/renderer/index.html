Exemplo de Arquivo para Importação em Lote

×

Para importar peritos em lote, crie um arquivo JSON com a seguinte estrutura:

[
  {
    "nome": "João Silva Santos",
    "cpf": "123.456.789-00",
    "ojs": [
      "1ª Vara do Trabalho de Campinas",
      "2ª Vara do Trabalho de Campinas",
      "Vara do Trabalho de Jundiaí"
    ]
  },
  {
    "nome": "Maria Oliveira Costa",
    "cpf": "987.654.321-11",
    "ojs": [
      "Vara do Trabalho de Atibaia",
      "LIQ2 - Jundiaí",
      "EXE1 - Campinas"
    ]
  }
]
Instruções:
  • O arquivo deve estar em formato JSON válido
  • Cada perito deve ter nome, cpf e ojs
  • O campo "ojs" deve ser um array com os órgãos julgadores
  • CPFs duplicados irão atualizar os dados existentes
  • Registros inválidos serão ignorados com aviso

Exemplo de Arquivo para Importação de Servidores em Lote

×

Para importar servidores em lote, crie um arquivo JSON com a seguinte estrutura:

[
  {
    "nome": "João Silva Santos",
    "cpf": "123.456.789-00",
    "perfil": "Assessor",
    "ojs": [
      "1ª Vara do Trabalho de Campinas",
      "2ª Vara do Trabalho de Campinas",
      "Vara do Trabalho de Jundiaí"
    ]
  },
  {
    "nome": "Maria Oliveira Costa",
    "cpf": "987.654.321-11",
    "perfil": "Secretário de Audiência",
    "ojs": [
      "Vara do Trabalho de Atibaia",
      "LIQ2 - Jundiaí",
      "EXE1 - Campinas"
    ]
  },
  {
    "nome": "Carlos Eduardo Ferreira",
    "cpf": "456.789.123-22",
    "perfil": "Administrador",
    "ojs": [
      "3ª Vara do Trabalho de Sorocaba",
      "Vara do Trabalho de Salto",
      "Vara do Trabalho de Itapetininga"
    ]
  }
]
Instruções:
  • O arquivo deve estar em formato JSON válido
  • Cada servidor deve ter nome, cpf, perfil e ojs
  • O campo "perfil" pode ser: Assessor, Secretário de Audiência, Administrador, Servidor
  • O campo "ojs" deve ser um array com os órgãos julgadores
  • CPFs duplicados irão atualizar os dados existentes
  • Registros inválidos serão ignorados com aviso
💡 Dica: Use o arquivo exemplo_importacao_servidores.json na raiz do projeto como referência.
